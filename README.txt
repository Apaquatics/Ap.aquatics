AP AQUATICS MINI WEBSITE

FILES
- index.html: page structure and links
- styles.css: colors, layout, buttons, and mobile styling
- script.js: bubbles and automatic copyright year
- assets/ap-aquatics-logo.jpeg: your logo

HOW TO EDIT
1. Open the folder in Visual Studio Code or another text editor.
2. Change wording and links inside index.html.
3. Change colors in styles.css under :root.
4. Replace the logo in assets while keeping the same filename.

HOW TO PUBLISH FREE
- GitHub Pages
- Cloudflare Pages
- Netlify

For GitHub Pages, upload all files while keeping the same folder structure.
Your QR code should point to the final published website URL.
